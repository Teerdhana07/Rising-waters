How It Works

Data-Driven Predictions

Rising Waters uses a trained XGBoost classifier that analyses six key meteorological parameters — annual rainfall, cloud coverage, and seasonal rainfall breakdowns — to determine flood probability for a given region.

The model was developed using historical flood and weather datasets, preprocessed with feature scaling, and evaluated against multiple algorithms to select the best-performing classifier for production deployment.
